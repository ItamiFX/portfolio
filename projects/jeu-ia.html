Implémentation des règles d'un jeu & IA

Ce projet est une implémentation en Java d'un jeu avec plusieurs niveaux et deux intelligences artificielles utilisant des stratégies différentes. L'objectif était d'explorer la conception d'IA simples, l'architecture orientée objet et les tests unitaires pour évaluer les performances des stratégies.

Fonctionnalités principales

  • Moteur de règles du jeu (gestion des états, tours, conditions de victoire).
  • Deux IA avec stratégies différentes (heuristique simple vs recherche approfondie).
  • Tests unitaires avec JUnit pour mesurer performance et robustesse.
  • Structure modulaire pour faciliter l'extension et les expérimentations.

Technologies

Java, JUnit, NetBeans, Git.

Comment exécuter

Le projet est une application Java — ouvrez-le dans votre IDE (NetBeans/IntelliJ) et exécutez la classe principale. Les tests JUnit peuvent être lancés depuis l'IDE ou via Maven/Gradle si vous avez le build script.
