Bot Discord – Automatisation de recherches

Projet personnel d’apprentissage à l’automatisation web : développement d’un bot Discord qui simule le comportement d’un utilisateur effectuant des recherches sur un site de petites annonces, afin de m’exercer à Selenium et à l’intégration avec Discord.

Objectifs du projet

  • Découvrir l’automatisation de navigation web avec Selenium.
  • Relier un script Python à un bot Discord pour envoyer des messages automatisés.
  • Simuler la recherche d’articles selon différents critères (prix, mot-clé, catégorie…).
  • Structurer un petit projet complet : configuration, exécution, logs, gestion d’erreurs simples.

Fonctionnement (simulation)

  • Le bot simule un utilisateur qui ouvre le site, renseigne des critères de recherche (par exemple marque, fourchette de prix), puis récupère les résultats affichés.
  • Les résultats simulés sont envoyés dans un salon Discord de test sous forme de messages (titre, prix, lien, etc.).
  • Le projet est utilisé uniquement dans un cadre d’apprentissage, sans usage réel pour des achats ou des actions sur le site.

Compétences mobilisées

  • Automatisation de navigation web (Selenium).
  • Interaction entre Python et l’API Discord.
  • Gestion de scripts, logs, variables d’environnement.
  • Organisation d’un projet personnel de bout en bout.

Technologies

Python, Selenium, API Discord, Git.
