Robot League (Symfony / MySQL)

Robot League est une plateforme web qui gère toutes les étapes d’une compétition de football pour robots : gestion sécurisée des utilisateurs, des équipes et des compétitions, avec une interface responsive

Le projet automatise aussi des tâches plus complexes comme la génération de créneaux et la planification des rencontres, afin de faciliter le travail des organisateurs

Fonctionnalités développées

  • Système sécurisé : gestion des utilisateurs, rôles et permissions
  • Création / gestion des équipes et des compétitions
  • Interface responsive avec Twig + TailwindCSS
  • Automatisation : génération de créneaux et planification des matchs
  • Gestion de données MySQL

Technologies

Symfony, PHP, MySQL, Twig, TailwindCSS, Git

Amélioration possible

  • Ajout d’un système de rencontres éliminatoires (phases finales / bracket).
