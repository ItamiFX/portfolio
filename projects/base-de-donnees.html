Création d'une base de données

Projet complet de conception et mise en place d’une base de données : définition du besoin, création du MCD, transformation en schéma relationnel et rédaction des scripts SQL.

Contexte

À partir d’un cahier des charges, il fallait modéliser les informations, identifier les entités, relations et contraintes, puis proposer une structure de base de données cohérente et performante.

Travail réalisé

  • Conception du MCD (Modèle Conceptuel de Données).
  • Passage au modèle logique et création du schéma relationnel.
  • Rédaction des scripts SQL de création (DDL) et d’insertion (DML).
  • Prise en compte des contraintes d’intégrité, des clés et de l’indexation.
  • Rédaction d’une courte documentation technique et d’un bilan.

Compétences mobilisées

  • Modélisation de données (MCD → relationnel).
  • SQL (création, insertion, intégrité référentielle).
  • Analyse de contraintes et choix de types.
  • Utilisation d’outils de conception (WinDesign).

Technologies

WinDesign, SQL Server / SQL, Git.
