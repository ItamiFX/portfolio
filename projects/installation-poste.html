Installation de poste

Projet d’installation et de configuration d’un poste virtuel léger conforme à un cahier des charges : choix du système, installation des outils de développement, automatisation et documentation.

Objectifs

  • Proposer une solution de poste de travail simple à déployer pour un étudiant en informatique.
  • Standardiser les outils (IDE, Git, Docker, etc.).
  • Automatiser un maximum d’étapes pour éviter les erreurs manuelles.

Aspects traités

  • Choix et configuration du système (Linux Mint).
  • Installation d’outils : IDE, Git, outils réseau, Docker, etc.
  • Écriture de scripts Bash pour automatiser l’installation.
  • Rédaction d’une documentation d’installation claire.

Technologies

Linux Mint, Bash, VMware, Git.
